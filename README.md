# MATLAB模板匹配车牌识别（含GUI界面，论文，详细注释）

#### 介绍
基于MATLAB的模板匹配车牌识别系统。含界面GUI。完美运行。流程是：读取车牌，灰度，二值化，边缘检测，车牌定位，字符切割，车牌识别。每个步骤都非常详细的注释。可二次开发成出入库识别，库外车牌播报预警，停车计费等。
